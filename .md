ROTA
    Method http - CRUD(Creat, Read, Update, Delete)
        GET - pega uma infor
        POST - cria um infor
        PUT - altera toda a infor
        PATH - altera parte da infor
        DELETE - apaga uma infor

    Name - Identificador de rota

    Function (Callback) - responsalvel por executar algum comando
